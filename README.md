# 📊 Amazon Sales Analytics Dashboard

Welcome to the first project of my professional Business Intelligence portfolio! This repository contains a fully interactive Power BI dashboard designed to transform raw retail sales data into strategic business insights.

## 🖼️ Dashboard Preview
![Amazon Sales Dashboard](dashboard_screenshot.jpg)

## 📌 Project Overview
The objective of this project was to clean, model, and visualize Amazon sales data to analyze revenue trends, evaluate product category performance, and monitor regional distribution.

## 🛠️ Core Technical Features & Architecture
* **Data Preparation & Querying:** Cleaned, structured, and normalized raw data using **Power Query**.
* **Robust Data Modeling:** Implemented a star schema layout by establishing direct relationships between the fact table and a custom **DAX Calendar Table** (Time Intelligence).
* **Advanced UI/UX Interactivity:** Engineered a hidden, sliding **Slicer Panel** using a combination of the Selection Pane and Bookmarks to save canvas space.
* **Contextual Business Insights:** Integrated **Visual Header Help Icons (`?`)** that deliver targeted executive summaries on hover, avoiding interface clutter.
* **Modern Formatting:** Applied conditional formatting, data-driven color gradients, and customized icons (`$`, `🛒`) to optimize reporting layout and visual scannability.

## 💡 Executive Insights Discovered
* **Revenue Stability:** Month-over-month sales trends indicate steady and consistent performance driven by regular market demand.
* **Category Performance:** Top-tier product categories dominate over 50% of the total revenue, highlighting key growth segments.
* **Market Presence:** Geographical sales distribution is highly balanced across all major territories, proving a well-diversified risk profile.
